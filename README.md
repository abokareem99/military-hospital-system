نظام المتابعة الرقمي - مستشفى السلاح الطبي ببورتسودان
​قسم النفسية
​هذا النظام مصمم لأتمتة وتنظيم سجلات المرضى في قسم النفسية بمستشفى السلاح الطبي بمدينة بورتسودان، مما يسهل عملية المتابعة الطبية وتحديث البيانات بشكل فوري وآمن.
​✨ المميزات الرئيسية
​إدارة سجلات المرضى: إضافة وتحديث بيانات المرضى بسهولة.
​نظام متابعة رقمي: استبدال السجلات الورقية بنظام إلكتروني كفء.
​واجهة مستخدم بسيطة: تعتمد على تقنيات الويب الحديثة (HTML/JavaScript) لضمان السرعة والسهولة.
​🛠 التقنيات المستخدمة
​Frontend: HTML5, CSS3, JavaScript.
​Database: Supabase 
​Deployment: استضافة عبر Vercel 
​Digital Tracking System - Military Hospital Port Sudan
​Psychiatric Department
​A digital solution developed to manage and track patient records at the Psychiatric Department of the Military Hospital in Port Sudan.
​✨ Key Features
​Patient Record Management: Streamlined interface to add and update patient information.
​Digital Tracking: Moving from paper-based archives to a secure digital registry.
​User-Friendly Interface: Built with clean HTML and JavaScript for efficient data entry.
​🛠 Tech Stack
​Frontend: HTML5, JavaScript.
​Backend: Database integration for secure record keeping.
